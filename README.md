# Treasure Island Text Adventure Game

A text-based "Choose Your Own Adventure" game built in Python. Players must navigate a mysterious island by making critical choices to find hidden treasure, avoiding dangerous traps, monsters, and unexpected deaths along the way.

## 🕹️ How to Play

The game starts automatically in your terminal. You will be prompted with scenarios and must type your choice to proceed:
* Type left or right to choose your path.
* Type swim or wait when encountering obstacles.
* Choose the right door color to claim the treasure!

## 🛠️ Tech Stack
* *Language:* Python 3
* *Concepts Used:* Conditional logic (if/elif/else), nested loops, user input handling, and text formatting.

## ⚙️ How To Run Locally

Follow these quick steps to play the game on your computer:

1. *Clone the repository:*
   bash
   git clone https://github.com
   

2. *Navigate into the project directory:*
   bash
   cd Treasure_Island_Project
   

3. *Run the game:*
   bash
   python task.py
